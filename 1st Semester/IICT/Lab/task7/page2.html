Tables  in  HTML

Tables

As normal tables , Tables in HTML also consists of rows and column
A table starts with the < table > and ends with < / table >
We can define Rows and column in the table and the number of these Rows and Column is programable
to define a row < tr > < / tr > these tags are used , " tr " stands for Table Row
to define a Column < td > < / td > these tags are used , " td " stands for table data
to define the header of the table or the field of the table the < th > < / th > tags are used , th stands for table header

Example

<table border="1" align=center>
<tr>
<th>  First Row  </th>
<th>  Second Row  </th>
<th>  Third Row  </th>
</tr>
<tr>
<td>  First Row , First Column  </td>
<td>  Second Row , Frist Column  </td>
<td>  Third Row , Frist Column  </td>
</tr>
<tr>
<td>  First Row , Second Column  </td>
<td>  Second Row , Second Column  </td>
<td>  Third Row , Second Column  </td>
</tr>
</table>

Will display the folowing table


  First Row     Second Row     Third Row  
  First Row , First Column     Second Row , Frist Column     Third Row , Frist Column  
  First Row , Second Column     Second Row , Second Column     Third Row , Second Column
